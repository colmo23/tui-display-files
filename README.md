# TUI File Viewer

A simple TUI application for viewing files in a directory, written in Go.

## Features

-   Lists all files in the current directory.
-   Navigate the file list with up/down arrow keys.
-   View file contents by pressing 'Enter'.
-   Scroll through file contents with PageUp/PageDown.
-   Return to the file list with 'Esc'.

## Build

To build the application, run the following command:

```bash
./build.sh
```

## Usage

To run the application, execute the binary:

```bash
./tui-display-files
```
